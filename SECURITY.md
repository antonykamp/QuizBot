# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in QuizBot, please report it privately rather than opening a public issue.

**Email:** info@antonykamp.de

Please include:
- A description of the vulnerability
- Steps to reproduce it
- Any potential impact

You can expect an initial response within 72 hours. We will work with you to understand and address the issue before any public disclosure.

## Scope

This policy applies to the QuizBot codebase and the live bot instance at [@antonysquizbot](https://t.me/antonysquizbot).
